# How To Create Your GitHub Page
[visit this demo](https://creatorcao.github.io/) 

1. Create a GitHub account

2. **Fork** this repository and change it to your name (yourname.github.io)
![](images/Fork.png)

3. Go to **Settings** --> **Pages** and **Choose a theme**
![](images/Settings.png)

4. Click *_config.yml* above to get started and fill in the details. Then click *index.md* and edit it to start creating your home page.
